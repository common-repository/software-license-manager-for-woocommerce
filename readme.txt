Generate a license key to in WooCommerce product with the Software License Manager.

== Description ==

= Issue a license key =
* Add the license key issued by [Software License Manager](https://wordpress.org/plugins/software-license-manager/) to the product sold by [WooCommerce](https://wordpress.org/plugins/woocommerce/).
* To activate the license key, you need code to client side.

= Cooperation with other plugins =
* Work with [WooCommerce](https://wordpress.org/plugins/woocommerce/).
* Work with [Software License Manager](https://wordpress.org/plugins/software-license-manager/).
* Work with [Software License Manager Client](https://wordpress.org/plugins/software-license-manager-client/).

= Example =
* [Create and sell license key in WooCommerce](https://blog.riverforest-wp.info/create-and-sell-license-key-in-woocommerce/)

== Installation ==

1. Upload `software-license-manager-for-woocommerce` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
